# Ai-project 

# Idea:-

# The project that I’m going to build is basically based on Print-on-Demand services. In this platform, we are building a system for manufacturers, where they can connect with multiple sellers.
# The idea is that when customers place orders with sellers, those orders will be visible to the sellers. After the seller approves the order, it will be pushed to our platform — the print-on-demand manufacturer platform.
# Then, as a manufacturer, whoever is using my platform will fulfill that order. This includes branding, printing, shipping, and everything related to order fulfillment.So, I want to create a platform around this entire workflow, and that’s why I’ve started this project.
